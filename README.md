# 🔐 Security Log Analyzer

A web-based cybersecurity tool built using Flask that analyzes log files to detect suspicious login attempts.

## 🚀 Features
- Upload log files
- Detect failed login attempts
- Extract IP addresses
- Count attack attempts
- Visualize data using charts

## 🛠 Tech Stack
- Python
- Flask
- HTML/CSS
- Bootstrap
- Chart.js

## 📊 Use Case
Useful for SOC analysts to monitor suspicious activity in logs.

## ▶️ Run Locally
```bash
pip install flask
python app.py
